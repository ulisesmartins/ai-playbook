# Git & GitHub for Designers

If you've ever stared at a terminal wondering why your changes disappeared, or panicked when GitHub said "merge conflict", you're not alone. This guide answers the most common questions designers ask about Git and GitHub.

Git is the tool that lets multiple people work on the same codebase without overwriting each other's work. GitHub is where that code lives online. Together, they're how we collaborate on UI Studio, Beacon, Dialtone, and every other repo at Dialpad.

## Find Your Level

The guide is split into levels, from setting up your account to advanced workflows. Each level has instructions for both **GitHub Desktop** and the **Terminal**.
