# Resources for UI Managers

The following section centralizes assets that are shared in different sections of the UI Confluence site. Here is a toolkit for UI Managers.

| Asset | Purpose / Summary |
|-------|-------------------|
| [Dialpad Managers Confluence](https://dialpad.atlassian.net/wiki/display/MGR) | Confluence site managed by People Ops — provides a range of useful resources for Dialpad Managers |
| [Visualizing UI Work](https://dialpad.atlassian.net/wiki/x/EAD4WQ) | A guide to how we visualize UI work using our project management tools |
| [Critiques and UI Reviews](https://dialpad.atlassian.net/wiki/x/D4AiKg) | Guidelines for critiques and design reviews |
| [UI Leadership Reviews](https://dialpad.atlassian.net/wiki/x/coDaIw) | A guide for how we handle UI Reviews with DLT, plus a tracker of those reviews |
| [Async UI Reviews](https://dialpad.atlassian.net/wiki/x/MgD1WQ) | Expectations for Designers and Managers on how to handle async reviews with Jira |
| [Product UI Job Levels](https://dialpad.atlassian.net/wiki/x/0AAbb) | Useful guide for Designers and Managers to assess leveling and align expectations on growth areas |
| [New Hire Onboarding Guide](https://dialpad.atlassian.net/wiki/x/AQAaFQ) | Step-by-step guide after the Recruiting team has extended an offer and before the new hire joins |
| [Various Documents & Templates](https://dialpad.atlassian.net/wiki/x/5ILzMQ) | A list of various documents to help you and designers expedite certain processes |
| [Expense Reimbursement](https://dialpad.atlassian.net/wiki/spaces/DES/pages/1181024426) | Guidelines for submitting expense reimbursements, along with policies and manager approval roles |
| [Paid Time Off (PTO)](https://dialpad.atlassian.net/wiki/spaces/DES/pages/725843994) | Guidelines for submitting PTO requests, along with policies |
