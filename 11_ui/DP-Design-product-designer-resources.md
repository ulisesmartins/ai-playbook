# Resources for Product Designers

This page serves as the go-to guide for all essential tools and utilities. Each entry includes the tool's name, description, use case, and owner.

These resources are also available at [dialpad.github.io/design-tools](https://dialpad.github.io/design-tools/).

## AI Agents

| Name | Description | When to use | Owner |
|------|-------------|-------------|-------|
| [Aerolabs Content Creator](https://chatgpt.com/g/g-8HaRxqqCY-aerolabs-content-creator) (GPT) | A specialized copywriter for creating realistic workplace dialogues for a fictional sustainable logistics company. | When designing or prototyping Dialpad UI. | Austen Ezzell |
| [Story Point Estimator](https://chatgpt.com/g/g-68c2d4df8d948191b22eff5918b2b13f-story-point-estimator-gpt) (GPT / Rovo) | Helps Designers and Engineers estimate effort using a consistent, story-point-based approach. | When assigning or reviewing story points for a Jira issue. | Belu Montoya, Tamara Paluch |
| [Product UI Feedback](https://chatgpt.com/g/g-68c17bc3571481918fa2154fa057c040-product-design-feedback) (GPT) | Gives candid, constructive, and brand-aligned feedback for design work in any stage. | When you want a critical review of a design. | Ted Goas |
| [Brand Brief Partner](https://chatgpt.com/g/g-68f29ae246d481918e6c647ff24c5ba7-brand-brief-gpt) (GPT) | Helps create a brief for the Brand UI team. | When you need to create or refine a marketing/creative brief. | Austen Ezzell |
| [Creative Partner](https://gemini.google.com/gem/036a9045ca03) (Gem) | Write content in Dialpad's brand voice. | When drafting a request submitted to the Brand UI Team. | Austen Ezzell |
| User Research Assistant (Rovo) | Supports Designers and PMs throughout the entire user research lifecycle. | When you need guidance to run a user research effort. | Guilla Colombres-Garmendia |
| Jira Work Item Writer (Rovo) | Creates clear Jira work item descriptions from templates. | When you need guidance to create and complete Jira work items. | Guilla Colombres-Garmendia |
| [UI Reviews Assistant](https://chatgpt.com/g/g-68c04cb15c4081918ae54025d861edc2-design-reviews-meeting-summary-assistant) (ChatGPT) | Summarize UI team meetings and extract actions from transcripts. | When you need a more detailed meeting recap. | Guilla Colombres-Garmendia |
