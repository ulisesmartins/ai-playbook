## How to use these files

- Reference the relevant file at the start of any session involving that tool
- Each file covers: what the tool is, core best practices, Dialpad-specific guidance, and common pitfalls
- Files are updated when major new features ship — check `last_updated` in the frontmatter
- For tool setup and MCP configuration, see `09_mcps/guides/`
- For role-specific tool recommendations, see the parent tool files in `08_tools/pm/`, `08_tools/engineering/`, `08_tools/design/`, `08_tools/cross-functional/`
