# Tools — UI

Curated AI tools for Designers at Dialpad across the design lifecycle. Start with **Glean + Gemini** for everyday work; layer Claude Code and UI Studio for structured design workflows.

<div class="grid cards" markdown>

-   :material-magnify:{ .lg .middle } **Glean**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Find prior design explorations, past briefs, research artifacts, competitive references, and component decisions before starting any new design. Cross-searches Drive, Confluence, and Jira.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-glean.md)

-   :material-google:{ .lg .middle } **Gemini for Workspace**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Summarize research notes, draft design rationale, write handoff annotations, generate alt text. Built into Gmail, Docs, and Drive — no context switching required.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-gemini-workspace.md)

-   :material-robot-excited:{ .lg .middle } **Claude Code**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    UI brief generation, component spec drafting, Dialtone design critique, and Gate 1 readiness checks via `/design-brief`, `/design-critique`, `/component-spec`, and `/gate-review-checklist`.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-claude-code.md)

-   :material-pencil-ruler:{ .lg .middle } **UI Studio**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    AI UI screen generation inside UI Studio from a text prompt. Remap generated screens to Dialtone DT9 components. Replaces external prototyping tools for most Gate 1 use cases.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-figma-make.md)

-   :material-test-tube:{ .lg .middle } **Maze / UserTesting**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Unmoderated usability testing on UI Studio prototypes. Run after Gate 1 feedback; export results as a research summary for the PRD Appendix and Gate 2 evidence.

    [:octicons-arrow-right-24: UI tools guide](DP-General-tools.md)

-   :material-chip:{ .lg .middle } **Dialtone MCP Server**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Gives Claude Code live access to Dialtone component API surface, token values, and usage examples. Improves accuracy on `/component-spec` and `/spec-to-code-context`.

    [:octicons-arrow-right-24: Setup guide](../../09_mcps/guides/DP-General-atlassian-setup.md)

</div>
