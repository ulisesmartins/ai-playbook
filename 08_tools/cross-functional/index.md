# Tools — Cross-functional

AI tools used across PM, Design, and Engineering at Dialpad. These tools span the full PDLC — from discovery through delivery — and are relevant regardless of role.

<div class="grid cards" markdown>

-   :material-robot-excited:{ .lg .middle } **Claude Code**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    The shared EPD execution layer. Runs skills, agents, and Cowork sessions from this repo. All EPD roles use Claude Code; the skills and context loaded differ by role.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-claude-code.md)

-   :material-magnify:{ .lg .middle } **Glean**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Cross-app internal knowledge base — searches Drive, Confluence, Jira, Salesforce, Zendesk, Monday, and GitHub in one query. The default starting point for all roles before any work session.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-glean.md)

-   :material-google:{ .lg .middle } **Gemini for Workspace**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Embedded AI in Google Workspace — email, Docs, Sheets, Slides, and Drive. The everyday writing layer for all EPD roles.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-gemini-workspace.md)

-   :material-ticket-outline:{ .lg .middle } **Atlassian Rovo**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    AI assistance inside Jira and Confluence — in-place doc Q&A, ticket creation, workflow automations. Shared by PM, Design, and Engineering for Atlassian-native work.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-atlassian-rovo.md)

-   :material-calendar-check:{ .lg .middle } **monday.com AI**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Project management AI across EPD — sprint planning, status rollups, task generation, and cross-functional board management within Monday.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-monday-ai.md)

-   :material-pencil-ruler:{ .lg .middle } **Design Studio**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    AI UI prototyping inside Design Studio used by both PM (Gate 1 mock screens) and Design (rapid exploration). Standard for all Dialtone-aligned prototype generation.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-figma-make.md)

-   :material-autorenew:{ .lg .middle } **n8n**
    <span class="dp-badge dp-badge--ga">GA</span>

    ---

    Cross-team workflow automation — Jira webhooks, Slack alerts, GitHub triggers, and custom EPD integration pipelines. Low-code; no infrastructure required.

    [:octicons-arrow-right-24: Best practices](../best-practices/DP-General-n8n.md)
