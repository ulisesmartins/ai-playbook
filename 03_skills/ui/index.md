# UI Skills

Claude Code slash commands for Designers. Critique designs, generate briefs, and document components — all from the CLI.

<div class="grid cards" markdown>

-   :material-image-search:{ .lg .middle } **UI Critique**

    ---

    Critique a design against Dialpad's design principles, Dialtone standards, and user experience best practices.

    **Command:** `/design-critique`

    [:octicons-arrow-right-24: View skill](DP-General-design-critique.md)

-   :material-pencil-ruler:{ .lg .middle } **UI Brief**

    ---

    Generate a structured design brief from a feature description or PRD excerpt.

    **Command:** `/design-brief`

    [:octicons-arrow-right-24: View skill](DP-General-design-brief.md)

-   :material-layers-outline:{ .lg .middle } **Component Spec**

    ---

    Document a UI component in Dialtone format — props, variants, accessibility notes, and usage guidelines.

    **Command:** `/component-spec`

    [:octicons-arrow-right-24: View skill](DP-General-component-spec.md)

</div>

---

!!! tip "How to install a skill"
    Copy the `.md` file into `~/.claude/commands/` (global) or `.claude/commands/` (project-local). Then run it with the slash command shown above.
