# PM Skills

Claude Code slash commands for Product Managers. Install once, run anywhere in your project.

<div class="grid cards" markdown>

-   :material-file-document-edit:{ .lg .middle } **Draft a PRD**

    ---

    Generate a complete, Dialpad-formatted PRD from a feature brief. Pulls context from your CLAUDE.md automatically.

    **Command:** `/draft-prd`

    [:octicons-arrow-right-24: View skill](DP-General-draft-prd.md)

-   :material-ticket-outline:{ .lg .middle } **Generate JIRA Epic**

    ---

    Convert a feature description into a formatted JIRA epic with acceptance criteria and sub-tasks.

    **Command:** `/generate-jira-epic`

    [:octicons-arrow-right-24: View skill](DP-General-generate-jira-epic.md)

-   :material-pencil-ruler:{ .lg .middle } **PM + UX Flow Review**

    ---

    Create and refine UI flows from descriptions or specs. Generates step-by-step flows with copy and edge states, or critiques existing flows through 13 UX lenses grounded in Dialpad's design system.

    **Command:** `/pm-ux-review`

    [:octicons-arrow-right-24: View skill](DP-General-pm-ux-review.md)

-   :material-magnify:{ .lg .middle } **Product Deep Dive**

    ---

    Analyze a product area with competitive context, user pain points, and strategic framing.

    **Command:** `/product-deep-dive`

    [:octicons-arrow-right-24: View skill](DP-General-product-deep-dive.md)

-   :material-lightbulb-outline:{ .lg .middle } **Prototype Brief**

    ---

    Create a prototype brief for a feature idea — scope, flows, and success criteria.

    **Command:** `/prototype-brief`

    [:octicons-arrow-right-24: View skill](DP-General-prototype-brief.md)

-   :material-note-text-outline:{ .lg .middle } **Release Notes from Ticket**

    ---

    Generate customer-facing release notes from a JIRA ticket or PR description.

    **Command:** `/release-notes-from-ticket`

    [:octicons-arrow-right-24: View skill](DP-General-release-notes-from-ticket.md)

-   :material-format-list-checks:{ .lg .middle } **Task Breakdown**

    ---

    Break a feature or epic into actionable, sized engineering tasks with clear acceptance criteria.

    **Command:** `/task-breakdown`

    [:octicons-arrow-right-24: View skill](DP-General-task-breakdown.md)

-   :material-brain:{ .lg .middle } **Brainstorm**

    ---

    Generate structured ideas and options for a product problem or opportunity.

    **Command:** `/brainstorm`

    [:octicons-arrow-right-24: View skill](DP-General-brainstorm.md)

</div>

---

!!! tip "How to install a skill"
    Copy the `.md` file into `~/.claude/commands/` (global) or `.claude/commands/` (project-local). Then run it with the slash command shown above.
