# Cross-functional Skills

Claude Code slash commands that span PM, Design, and Engineering — from gate reviews to GTM drafts and on-call triage.

<div class="grid cards" markdown>

-   :material-check-decagram:{ .lg .middle } **Gate Review Checklist**

    ---

    Run a structured design gate review checklist against a feature spec, PRD, or design file.

    **Command:** `/gate-review-checklist`

    [:octicons-arrow-right-24: View skill](DP-General-gate-review-checklist.md)

-   :material-rocket-launch-outline:{ .lg .middle } **GTM Draft**

    ---

    Draft a go-to-market plan from a PRD or launch brief — messaging, channels, timing, and owner table.

    **Command:** `/gtm-draft`

    [:octicons-arrow-right-24: View skill](DP-General-gtm-draft.md)

-   :material-phone-alert:{ .lg .middle } **On-Call Triage**

    ---

    Triage an on-call incident — summarize signals, identify probable cause, and generate a comms draft.

    **Command:** `/on-call-triage`

    [:octicons-arrow-right-24: View skill](DP-General-on-call-triage.md)

-   :material-calendar-clock:{ .lg .middle } **Daily Workflow**

    ---

    Automatic session notes, structured morning briefings, end-of-day wraps, weekly snippets, and action-item tracking across your team.

    **Commands:** `/day-start`, `/day-wrap`, `/wrap`, `/snippet`, `/reflect`

    [:octicons-arrow-right-24: View skill](DP-General-daily-workflow.md)

</div>

---

!!! tip "How to install a skill"
    Copy the `.md` file into `~/.claude/commands/` (global) or `.claude/commands/` (project-local). Then run it with the slash command shown above.
