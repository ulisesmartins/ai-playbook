# Cowork Plugins — Design

Cowork task definitions for Designers. Each plugin is a task description you delegate to Cowork — Claude executes it autonomously and delivers structured results.

<div class="grid cards" markdown>

-   :material-palette-swatch:{ .lg .middle } **Design Review Team**

    ---

    A delegated task that reviews a design from three perspectives: visual quality, Dialtone compliance, and product/UX clarity. Output is a consolidated critique with severity-ranked gaps.

    [:octicons-arrow-right-24: View plugin](DP-General-design-review-team.md)

-   :material-check-decagram:{ .lg .middle } **Dialtone Audit Team**

    ---

    A delegated task that audits a component or screen for Dialtone token usage, accessibility compliance, and component pattern consistency.

    [:octicons-arrow-right-24: View plugin](DP-General-dialtone-audit-team.md)

</div>

---

!!! info "How to use a Cowork plugin"
    Paste the plugin task description into Cowork or Claude Code. See the [Getting Started guide](../../06_cowork-plugins/DP-General-cowork-getting-started.md) for instructions.
