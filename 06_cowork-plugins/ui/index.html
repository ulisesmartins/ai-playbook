    Cowork Plugins — UI¶

    Cowork task definitions for Designers. Each plugin is a task description you delegate to Cowork — Claude executes it autonomously and delivers structured results.

    • UI Review Team

      A delegated task that reviews a design from three perspectives: visual quality, Dialtone compliance, and product/UX clarity. Output is a consolidated critique with severity-ranked gaps.

      View plugin

    • Dialtone Audit Team

      A delegated task that audits a component or screen for Dialtone token usage, accessibility compliance, and component pattern consistency.

      View plugin

    How to use a Cowork plugin

    Paste the plugin task description into Cowork or Claude Code. See the Getting Started guide for instructions.
