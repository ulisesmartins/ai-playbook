# Cowork Plugins — Coding

Cowork task definitions for Engineers. Each plugin is a task description you delegate to Cowork — Claude executes it autonomously and delivers structured results.

<div class="grid cards" markdown>

-   :material-source-branch:{ .lg .middle } **Code Review Team**

    ---

    A multi-agent code review team — one agent checks logic, one checks test coverage, one checks Dialpad coding standards.

    [:octicons-arrow-right-24: View plugin](DP-General-code-review-team.md)

-   :material-bug:{ .lg .middle } **Debug Team**

    ---

    A multi-agent debugging team that analyzes a bug report, traces probable causes, and proposes fixes with test coverage.

    [:octicons-arrow-right-24: View plugin](DP-General-debug-team.md)

-   :material-test-tube:{ .lg .middle } **Test Coverage Team**

    ---

    A delegated task that audits test coverage on a module and generates missing unit, integration, and edge-case tests.

    [:octicons-arrow-right-24: View plugin](DP-General-test-coverage-team.md)

</div>

---

!!! info "How to use a Cowork plugin"
    Paste the plugin task description into Cowork or Claude Code. See the [Getting Started guide](../../06_cowork-plugins/DP-General-cowork-getting-started.md) for instructions.
