# Cowork Plugins — PM

Cowork task definitions for Product Managers. Each plugin is a task description you delegate to Cowork — Claude executes it autonomously and delivers structured results.

<div class="grid cards" markdown>

-   :material-file-document-check:{ .lg .middle } **PRD Review Team**

    ---

    A delegated task that reviews your PRD — one agent checks completeness, one checks assumptions, one generates open questions.

    [:octicons-arrow-right-24: View plugin](DP-General-prd-review-team.md)

-   :material-format-list-bulleted:{ .lg .middle } **Spec Breakdown Team**

    ---

    A delegated task that breaks a PRD into a full JIRA-ready epic with stories, estimates, and cross-functional dependencies.

    [:octicons-arrow-right-24: View plugin](DP-General-spec-breakdown-team.md)

-   :material-check-decagram:{ .lg .middle } **Support QA Review Team**

    ---

    A delegated task scoped to the Support product area that reviews a feature for QA readiness and support impact.

    [:octicons-arrow-right-24: View plugin](Support-qa-review-team.md)

</div>

---

!!! info "How to use a Cowork plugin"
    Paste the plugin task description into Cowork or Claude Code. See the [Getting Started guide](../../06_cowork-plugins/DP-General-cowork-getting-started.md) for instructions.
